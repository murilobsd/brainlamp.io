# Website BrainLamp.IO

<img src="app/images/logo.png" alt="brainlamp logo">

## Install

```shell
$ bower install
$ npm install
$ npm install -g gulp
```

## Run

Develop mode
```
$ gulp serve
```
**open browser: http://localhost:9000**

Production mode
```
$ gulp serve:dist
```

Build
```bash
$ gulp build
```

## Deploy to gh-pages

```
$ gulp deploy
```
